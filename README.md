# Ping Server

A very simple ping server developed using Spring Boot.

## Pre-requisites

- Java 17+
- Gradle
- Your favorite IDE

## Build

from the root folder of the project, execute

```console
gradle clean build
```

or

```console
./gradlew clean build
```

## Run

```console
gradle bootRun
```

or

```console
./gradlew bootRun
```
